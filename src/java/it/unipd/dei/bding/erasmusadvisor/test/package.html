Stand-alone classes for Test purposes. Contains all of the classes, executable "stand-alone", used for testing package beans, and database resources. Although this package is not in any way necessary for the non-functioning of the application, it is still essential in development and debugging of the layers below the Servlets. The use of the single DriverManager instead of a DataSource for connections, eliminates the need to run the entire webserver. @author Alessandro Beltramin @author Mauro Piazza @author Luca Schiappadini @author Nicola Tommasi @since 1.00
